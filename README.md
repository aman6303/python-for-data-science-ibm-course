# Python for Data Science - IBM Course

This repository contains materials, notebooks, and assignments for the **Python for Data Science, AI & Development** course by IBM (available on Coursera/edX). It covers the fundamental concepts of Python programming and its application in Data Science, including data structures, logic, file handling, and API interaction.

## Repository Structure

The course material is organized into modules, each focusing on a specific aspect of Python programming for data science:

| Module | Directory | Description |
| :--- | :--- | :--- |
| **Module 1** | `module1_python_basics` | Introduction to Python syntax, types, expressions, and variables. |
| **Module 2** | `module2_data_structures` | In-depth look at Lists, Tuples, Dictionaries, and Sets. |
| **Module 3** | `module3_fundamentals` | Python programming fundamentals including Loops, Functions, and Exception Handling. |
| **Module 4** | `module4_working_with_data` | Reading/Writing files and analyzing data using Pandas and NumPy. |
| **Module 5** | `module5_api_and_datacolection` | Working with APIs (Simple APIs, REST APIs) and web scraping basics. |
